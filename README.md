Introduction

Car companies generate large volumes of sales data every day. However, when this data is stored 
in multiple spreadsheets or CSV files, it becomes difficult to analyze trends and derive meaningful insights. 
The goal of this project is to design a scalable analytics pipeline that consolidates sales data, ensures data quality, and provides interactive dashboards for decision-makers.
Tools and Technologies used : 
Python (pandas, numpy, sqlite3)
SQLite Database
Power BI Desktop
CSV datasets
Visual Studio Code / Jupyter Notebook
